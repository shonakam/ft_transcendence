# チャットシステム 画面仕様 (UI/UX)

## 1. 画面遷移図 (Screen Flow)
ユーザーがチャット機能を利用する際の流れです。

```{mermaid}
graph TD
    A[メインダッシュボード] -->|チャットアイコンをクリック| B[チャット画面]

    B --> C{ルーム選択}
    C -->|任意のルームを選択| D[チャット表示]

    D --> F[メッセージ送信/閲覧]

    F -->|ユーザー名をクリック| G[ユーザープロフィール/アクションメニュー]
    G -->|DMを開く| H[DMチャット表示]
    G -->|ブロック| I[ブロック完了/リスト更新]
    G -->|ゲームに招待| J[招待URLを送信]
```

## 2. 各画面の詳細仕様

### 2.1 ルーム一覧 (SideBar)
- **Global Chat**: 常に最上部に固定。
- **DMリスト**: 直近でやり取りのあった順に表示。

### 2.2 メッセージエリア (MainChat)
- **自分/相手の区別**: 吹き出しの左右配置。
